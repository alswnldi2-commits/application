# SAT 영어 단어 학습 사이트 📚

Streamlit을 이용한 SAT 빈출 영어 단어 학습 웹 애플리케이션입니다.

## 주요 기능

- 📇 **단어 목록**: 100개의 SAT 빈출 단어를 한눈에 확인
- 🃏 **플래시카드**: 카드 형식으로 단어를 학습하며 뜻과 예문 확인
- ✏️ **퀴즈**: 단어 → 뜻, 뜻 → 단어 두 가지 모드로 실력 테스트
- 🔍 **검색**: 단어나 뜻으로 빠르게 검색
- ⭐ **즐겨찾기**: 중요한 단어를 저장하여 나중에 다시 학습
- 📊 **학습 통계**: 학습 진행도와 통계를 실시간으로 확인

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
streamlit run app.py
```

3. 브라우저에서 자동으로 열리는 페이지에서 학습을 시작하세요!

## 사용 방법

### 홈
- 애플리케이션의 개요와 사용 방법을 확인할 수 있습니다.

### 단어 목록
- 모든 단어를 목록 형식으로 확인
- 검색 기능으로 원하는 단어 찾기
- 즐겨찾기 및 학습 완료 표시

### 플래시카드
- 카드 형식으로 단어 학습
- 이전/다음 버튼으로 이동
- 랜덤 단어 선택 기능
- 학습 진행도 표시

### 퀴즈
- 5~20개의 문제로 구성된 퀴즈
- 단어 → 뜻, 뜻 → 단어 두 가지 모드
- 정답률과 상세 결과 확인
- 정답한 단어는 자동으로 학습 완료 처리

### 검색
- 단어나 뜻으로 빠른 검색
- 검색 결과에서 바로 즐겨찾기 및 학습 완료 표시

### 즐겨찾기
- 저장한 단어들을 모아서 확인
- 필요 없으면 제거 가능

```
pythonproject/

├── requirements.txt    # 필요한 Python 패키지
└── README.md          # 사용 설명서
```

## 기술 스택

- **Streamlit**: 웹 애플리케이션 프레임워크
- **Python 3.7+**: 프로그래밍 언어
- **JSON**: 단어 데이터 저장 형식

## 특징

- 🎨 **아름다운 UI**: 그라데이션과 카드 디자인으로 시각적으로 매력적인 인터페이스
- 💾 **세션 상태 관리**: 학습 진행도와 즐겨찾기가 브라우저 세션 동안 유지
- 📈 **실시간 통계**: 학습 진행도를 실시간으로 확인
- 🔄 **반응형 디자인**: 다양한 화면 크기에 대응

## 커스터마이징

`sat_words.json` 파일을 수정하여 단어를 추가하거나 수정할 수 있습니다. 각 단어는 다음 형식을 따릅니다:

```json
{
  "word": "단어",
  "meaning": "뜻",
  "example": "예문"
}
```
